Stuff used by Symbosite

As mentioned on the landing page, the backend of Symbosite is written in python. It would have probably been better to write it in something like ASP.Net or Javascript, but that would have been both a: sensible, and b: boring.

We disaprove of boringness, and being sensible is for sensible people.

Python libaries

Other than the base python library, this site also uses a number of other things. These include:

  • SCSS - For processing the style sheets (and why there is some fanciness on the site)
  • Pygments - For all the fancy syntax hilighting.
  • Sympy - For processing symbolic algebra and doing fancy stuff like that
  • Numpy - For bulk mathematical processing
  • Matplotlib - For plotting stuff like graphs
