About

This site was made by shutebt01 and jediminer543 as part of a project to encourage people to do maths stuff on the computer. Why? Because lots of tools that do this are either hard to use, or expensive, or both. As such, I decided to try and write a simpler one myself.

As for if it works; the fact that someone is reading this means it probably did. Or maybe it didn't. Who knows.

Contributing

First things first: if anyone wants to sugguest a better style for the site, let us know. Neither of us are graphic designers, nor do we have much of an aethetic ability, so this site probably looks terrible, and we havent realised yet.

If you wish to sugguest a feature, or highlight a bug, head over to github create an issue. If you wish to fix said bug/add said feature, create a pull request.
