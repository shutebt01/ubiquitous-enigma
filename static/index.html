What is Symbosite

Symbosite is a site designed by a us, to make it easier for not-programmers, or just those who don't want to program, to use computational libraries. It also includes help files to assist those who want to learn to use computational libaries. Its name is derived from Symbol, and Site, which should be unsuprising to anyone who is paying attention (which you, good reader probably are).

This site will be written in a half serious style, including randomly talking to the reader (Hi BTW). The aim of this is to make the content more engaging. And becuase it is a style we can both write in. But we're totally doing it becuase it is easier to read.

This site consists of two main features: A maths and dataprocessing section, and helpsheets, on both computational stuff and maths. The maths stuff is under construction (I.e. there but hugely broken) whereas the computational stuff like dataprocessing is NYI.

For the more technically minded people, this site is writen with a frontend of html, which you should have guessed, since your using a browser (I assume; if your some reader from the future viewing this some other way then hi). It is using the default 

http.server
as the backend, albeit with lots of custom code.

If you feel like adding something, pointing out a spelling mistake, or just requesting a feature, don't hesitate to ask on Github.
