TeX

Tl;Dr

Tex is a language in which text is formatted with functions begining with a `\`. LaTeX is an extention adding maths functions. Links: To Examples

What is TeX

TeX is a typesetting system designed and mostly written by Donald Knuth and released in 1978. - Wikipedia

This is a pretty good sumary of what tex is. While the wikipedia article will go into far more detail the function of TeX can be sumarised as follows

  • Text input is processed as text
  • Functions start with a 
    \
  • These functions are used to alter the output
  • New functions can be defined
TeX on its own is very usefull, but is far more usfull when used in conjunction with stuff others have already done. Hence enter:

LaTeX

Latex is command set for TeX that adds more commands, and is very usefull for maths, including things such as square roots 

\sqrt{x}
and fractions 
\frac{x}{y}
. If you are curious about the full feature set Here is a good wikibook including a large number of examples. But now onto our own examples which will be far more compresed.
