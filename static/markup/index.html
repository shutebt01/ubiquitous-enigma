Markup

What is a Markup language

Markup languages are languages used to represent stuff for computers to process. Thhis page is written in one such mark up language.

Why am I bothering covering Markup languages

Markup languages are great for people trying to do maths on the electronics, as it gives a form to said maths that both people and computers can understand

For example, all the maths on this site that I write are writen in $\LaTeX$, which is a subset of the $\TeX$ markdown language. This allows me to use fancy maths stuff such as $x \in \mathbb{N}$ and ${(\frac{1}{x})}^{x}$ without needing hack it into working. Technically I should be using MathML, which is far easier for computers to understand, but is also far harder for humans to use; this is also why you likley saw the Commands for the maths being displayed while the page loaded.
